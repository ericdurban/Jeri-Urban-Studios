How Payments Work

Session Frequency: Each session is a half-hour in length, once a week. Please refer to the tables below to see how much an individual session costs.

Piano

$15 per session

Flute

$25 per session

Payment Frequency: Payments are due monthly, with the first payment required at the start of each calendar month.

How to Calculate Monthly Payments: At the beginning of each month, refer to the calendar and count the total number of session days. You'd then multiply that number by your session cost. If you have any prior commitments, please inform me in advance and then subtract those dates from your total sessions for the month. Any missed sessions beyond that will not be refunded.

Example: (Piano sessions= $15) * (Total sessions for calendar month=4)= $60 owed on first calendar session.

In the event that someone is sick, we can attempt a make-up. However, please know this is not guaranteed.
